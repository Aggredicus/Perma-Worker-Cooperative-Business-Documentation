# Decision Log

## Purpose

This log is the official running register of major governance, legal, tax, compensation, policy, and structural decisions. It should be updated whenever the Company adopts a significant document, changes a material policy, modifies governance, or approves a major transaction.

## Instructions for use

1. Create one entry per decision.
2. Link the decision to the source document and any meeting minutes or written consent.
3. Record who approved the decision and when it became effective.
4. Do not delete old entries; instead add superseding entries.
5. Where possible, store signed evidence outside the repo and note the file location.

## Status codes

- **Proposed** — drafted but not approved
- **Approved** — authorized but not yet implemented
- **Effective** — in force
- **Superseded** — replaced by a later decision
- **Retired** — no longer used
